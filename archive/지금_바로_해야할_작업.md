## 🎯 2단계로 끝내기

### ✅ Step 1: Column Order 정리 (1분)

```
□ Column Order 섹션
□ 체크 해제할 필드:
   ❌ 관련파일
   ❌ 분양가, 발코니, 옵션비, 프리미엄, 합계
   ❌ 전세가, 보증금, 월세, 관리비
   ❌ 매물설명, 입주가능일, 방향, 만기예정일
   ❌ 접수자, 접수일
   ❌ 기타 필요 없는 것들

□ 체크 유지할 필드:
   ✅ 단지명
   ✅ 동
   ✅ 호
   ✅ 타입
   ✅ 거래유형
   ✅ 거래상태 (선택)
   ✅ 고객

□ Save
```

---

### ✅ Step 2: 단지명 Valid_If 설정 (30초)

```
□ 우측 미리보기에서 "단지명" 필드 클릭

□ 아래로 스크롤 → Advanced settings

□ Valid If 입력:
   IN([단지명].[단지명], ["아산배방1단지우방아이유쉘", "아산배방우방아이유쉘2단지"])

□ Save
```
