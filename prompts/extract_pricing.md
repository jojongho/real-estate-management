# 분양가 정보 추출 프롬프트

당신은 아파트 입주자 모집공고문 PDF에서 분양가 정보를 추출하는 전문가입니다.

## 📋 작업 지침

첨부된 PDF 파일에서 **타입별 × 층별 분양가 및 납부 일정 정보**를 추출하여 JSON 형식으로 출력하세요.

## 🎯 추출해야 할 정보

각 타입(평형) × 층 조합별로 다음 **모든** 금액 정보를 추출:

1. **타입** (예: 84A, 101A, 105, 119B)
2. **층구분** (예: 2층, 3층, 5~9층, 10층 이상)
3. **대지비** (원 단위 숫자만, 콤마 제외)
4. **건축비** (원 단위 숫자만, 콤마 제외)
5. **부가가치세** (원 단위 숫자만, 콤마 제외)
6. **분양가** (대지비 + 건축비 + 부가가치세)
7. **1차 계약금** (원 단위 숫자만)
8. **2차 계약금** (원 단위 숫자만)
9. **중도금 1회** (원 단위 숫자만)
10. **중도금 2회** (원 단위 숫자만)
11. **중도금 3회** (원 단위 숫자만)
12. **중도금 4회** (원 단위 숫자만)
13. **중도금 5회** (원 단위 숫자만)
14. **중도금 6회** (원 단위 숫자만)
15. **잔금** (원 단위 숫자만)

## ⚠️ 주의사항

1. **타입과 층별로 구분**: 같은 타입이라도 층마다 금액이 다를 수 있으니 **층별로 모두 추출**
2. **모든 금액 항목 필수**: 대지비부터 잔금까지 **15개 항목 모두 추출** (누락 시 0으로 표시)
3. **숫자만 추출**: 모든 금액은 숫자만 추출 (콤마, "원", "만원" 등 제외)
4. **층 표현**:
   - 단일 층: "3층", "5층" 형태
   - 범위: "5~9층", "10층 이상" 형태 유지
5. **정확성 우선**: 불확실한 데이터는 포함하지 마세요
6. **JSON만 출력**: 설명 없이 순수 JSON만 출력

## 🔍 데이터 검증

추출 전 반드시 확인:
- [ ] 모든 타입의 분양가가 포함되었는가?
- [ ] 각 타입의 **모든 층**에 대한 정보가 있는가?
- [ ] 15개 금액 항목이 **모두** 추출되었는가?
- [ ] 금액이 올바른가? (분양가 = 대지비 + 건축비 + 부가세)
- [ ] 계약금 + 중도금 + 잔금 = 분양가 인가?

## 💡 중도금 처리

- PDF에 중도금이 **6회 미만**으로 표시된 경우: 없는 회차는 0으로 표시
- PDF에 중도금이 **6회 초과**로 표시된 경우: 처음 6회만 추출
