# 구글 시트 건축물대장 자동화 설정 가이드

Vworld API 연결 불안정으로 인해, 보다 안정적인 **구글 시트(Google Sheets)** 기반 자동화로 전환하기 위한 가이드입니다.

## 1. 구글 시트 준비

새 구글 스프레드시트를 생성하고, 첫 번째 행(헤더)에 다음 컬럼명을 정확히 입력하세요.

| A | B | C | D | E |
| :--- | :--- | :--- | :--- | :--- |
| **주소** | **도로명주소** | **일반건물여부** | **대지면적** | **건축면적** |

| F | G | H | I | J |
| :--- | :--- | :--- | :--- | :--- |
| **건폐율** | **연면적** | **용적률산정연면적** | **용적률** | **구조코드명** |

| K | L | M | N |
| :--- | :--- | :--- | :--- |
| **기타구조** | **사용승인일** | **내진설계여부** | **내진능력** |

> **팁**: '주소' 컬럼에 지번 주소를 입력하면 자동화가 시작됩니다.

## 2. n8n 워크플로우 설정

이미 시스템에 저장된 구글 시트용 워크플로우 파일을 사용합니다.

1.  **파일 위치**: `- System/07. Workflows/n8n/n8n건축물대장조회.json`
2.  **Import**: n8n 화면 우측 상단 `...` → `Import from JSON` → 위 파일 내용 복사/붙여넣기
3.  **노드 설정**:
    *   **Google Sheets Trigger**:
        *   `Credential`: 본인의 구글 계정 선택
        *   `Document`: 위에서 만든 시트 선택
        *   `Sheet`: 시트 이름 선택 (보통 '시트1')
    *   **HTTP Request - Vworld**: `Key` 입력
    *   **Node 3, 3-1 (건축물대장 조회)**: `Service Key` 입력 (공공데이터포털 디코딩된 키)
    *   **Update row in sheet**:
        *   `Document` & `Sheet`: 위와 동일하게 선택

## 3. 실행 테스트

1.  n8n 워크플로우를 **Active**로 켭니다.
2.  구글 시트 '주소' 칸에 테스트할 주소를 입력합니다 (예: `충청남도 천안시 서북구 불당동 1329`).
3.  약 1분 이내에 나머지 칸들이 자동으로 채워지는지 확인합니다.

---
**참고**: n8n이 Railway 서버(해외 IP)에 있어 Vworld API 연결이 간혹 끊길 수(`ECONNRESET`) 있습니다. 이 경우 n8n의 `Retry on Fail` 옵션을 켜두면 도움이 될 수 있습니다.
