# Slice 사용법 - 30초 요약 ⚡

**목적**: 특정 폼에서만 단지명 2개만 보이게

---

## 🚀 3단계

### 1. Slice 만들기 (10초)
```
□ Data → Slices → + New Slice
□ Name: 배방아이유쉘_전용
□ Table: 아파트매물
□ Condition: IN([단지명].[단지명], ["아산배방1단지우방아이유쉘", "아산배방우방아이유쉘2단지"])
□ Save
```

### 2. 폼에 적용 (10초)
```
□ UX → Views → 배방아이유쉘_고객접수 선택
□ Detail slice: 배방아이유쉘_전용 선택
□ Save
```

### 3. 완료! (10초)
```
□ Preview 클릭
□ 단지명 2개만 표시되는지 확인 ✅
```

---

## 💡 핵심 개념

```
Slice = 데이터 필터
   ↓
폼에 Slice 적용 = 폼 전체가 필터링됨
   ↓
단지명 드롭다운도 자동으로 필터링됨 ✅
```
