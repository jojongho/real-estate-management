# 📁 데이터 폴더 구조

이 폴더는 입주자모집공고 PDF → Excel 정규화 파이프라인의 데이터를 관리합니다.

## 📂 폴더 설명

```
data/
├── raw/                    # 원본 파일 (수정하지 않음)
│   ├── pdf/               # 📄 입주자모집공고 PDF 파일
│   └── excel/             # 📊 원본 Excel/CSV 데이터
├── processed/             # ✅ 전처리된 중간 데이터
├── output/                # 🎯 최종 결과물 (정규화된 Excel)
└── samples/               # 📋 테스트용 샘플 데이터
```

## 📥 파일 넣는 방법

1. **입주자모집공고 PDF**: `raw/pdf/` 에 넣기
2. **참고용 Excel 파일**: `raw/excel/` 에 넣기
3. **테스트용 샘플**: `samples/` 에 넣기

## ⚠️ 주의사항

- `raw/` 폴더의 원본 파일은 절대 수정하지 마세요
- 처리 결과는 `processed/` 또는 `output/`에 저장됩니다
- `.gitignore`로 대용량 파일은 Git에서 제외됩니다
