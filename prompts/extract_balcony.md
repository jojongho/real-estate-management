# 발코니 확장 정보 추출 프롬프트

## 🎯 추출해야 할 정보

각 타입(평형)별로 다음 정보를 추출:

1. **타입** (예: 84A, 105, 119B)
2. **발코니 확장 금액** (원 단위 숫자만, 콤마 제거)
3. **확장 가능 여부** (가능/불가능)
4. **비고** (예: 필수선택, 선택옵션 등)

## 📊 출력 형식

다음과 같은 JSON 형식으로만 응답하세요:

```json
{
  "발코니": [
    {
      "타입": "84A",
      "확장금액": 15000000,
      "확장가능": "가능",
      "비고": "선택옵션"
    },
    {
      "타입": "105",
      "확장금액": 18000000,
      "확장가능": "가능",
      "비고": "필수선택"
    }
  ]
}
```

## 🔍 추출 가이드라인

### 금액 추출 규칙
- "15,000,000원" → 15000000 (콤마와 '원' 제거)
- "1500만원" → 15000000 (만원 단위를 원 단위로 변환)
- 금액이 명시되지 않은 경우: 0

### 확장 가능 여부
- 발코니 확장 가능한 경우: "가능"
- 발코니 확장 불가능한 경우: "불가능"
- 명시되지 않은 경우: "가능" (기본값)

### 비고 정보
- "필수선택", "선택옵션", "계약시선택" 등의 정보 추출
- 특별한 조건이나 제한사항이 있으면 기록
- 없으면 빈 문자열 ""

## ⚠️ 중요사항

1. **반드시 위의 JSON 형식으로만 응답하세요**
2. 다른 설명이나 텍스트는 포함하지 마세요
3. 모든 타입에 대한 발코니 정보를 빠짐없이 추출하세요
4. 금액은 반드시 숫자(정수)로 변환하세요
5. 발코니 정보가 없는 경우 빈 배열 `{"발코니": []}`을 반환하세요
