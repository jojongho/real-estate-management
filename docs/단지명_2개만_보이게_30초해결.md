# 단지명 2개만 보이게 - 30초 해결! ⚡

**문제**: Slice 적용해도 드롭다운에 전체 목록이 나타남

---

## 🎯 원인

```
Slice는 아파트매물 데이터를 필터링
단지명 드롭다운은 아파트단지 테이블을 참조
→ 별개! ❌
```

---

## ✅ 해결: Form에서 Valid_If 설정

**30초 작업:**

```
1. UX → Views → 배방아이유쉘_고객접수 폼 편집

2. 단지명 필드 클릭

3. 아래로 스크롤 → "Advanced settings"

4. Valid If에 입력:
   IN([단지명].[단지명], ["아산배방1단지우방아이유쉘", "아산배방우방아이유쉘2단지"])

5. Save

끝! ✅
```

---

## 🧪 테스트

```
□ 폼 열기

```
