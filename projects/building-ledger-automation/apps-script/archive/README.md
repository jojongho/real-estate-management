## 🔄 현재 사용 중인 스크립트

- **역할**: 메인 매물관리 스크립트
- **기능**:
  - 하이브리드 ID 생성 (수식 미리보기 + 영구 저장)

  - 자동 데이터 불러오기 (onEdit 트리거)
  - 필수 필드 검증 강화
  - 중복 등록 방지

### 2. `option.js`
- **역할**: 옵션 데이터 관리
- **기능**:
  - 외부 통합단지DB에서 옵션 데이터 가져오기
  - 타입별 옵션 필터링 (정확 일치 + 전체 + 다중 타입)
  - 발코니확장 제외 처리
  - 체크박스 UI 자동 생성
