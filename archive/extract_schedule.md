# 단지 일정 추출 프롬프트

당신은 아파트 입주자 모집공고문 PDF에서 단지 일정 정보를 추출하는 전문가입니다.

## 📋 작업 지침

첨부된 PDF 파일에서 **단지 일정 정보**를 추출하여 JSON 형식으로 출력하세요.

## 🎯 추출해야 할 정보

모든 주요 일정을 다음 정보로 추출:

1. **일정명** (예: 입주자모집공고일, 특별공급, 1순위, 당첨자발표, 계약체결, 1차계약금, 2차계약금, 중도금1회, 잔금)
2. **시작일** (YYYY. M. D 형식, 예: 2025. 6. 15)
3. **종료일** (YYYY. M. D 형식, 예: 2025. 6. 20)

## 📊 출력 형식

```json
{
  "일정": [
    {
      "일정명": "입주자모집공고일",
      "시작일": "2024. 10. 4",
      "종료일": "2024. 10. 4"
    },
    {
      "일정명": "특별공급",
      "시작일": "2024. 10. 14",
      "종료일": "2024. 10. 14"
    },
    {
      "일정명": "1순위",
      "시작일": "2024. 10. 15",
      "종료일": "2024. 10. 15"
    },
    {
      "일정명": "당첨자발표",
      "시작일": "2024. 10. 22",
      "종료일": "2024. 10. 22"
    },
    {
      "일정명": "계약체결",
      "시작일": "2024. 11. 4",
      "종료일": "2024. 11. 6"
    },
    {
      "일정명": "1차계약금",
      "시작일": "2024. 11. 4",
      "종료일": "2024. 11. 6"
    },
    {
      "일정명": "2차계약금",

    {

    {
      "일정명": "잔금",
      "시작일": "2025. 6. 30",
      "종료일": "2025. 6. 30"
    }
  ]
}
```

## ⚠️ 주의사항

1. **청약 일정**: 공고일, 특별공급, 1순위, 2순위, 당첨자발표, 계약체결 등
2. **납부 일정**: 계약금(1차/2차), 중도금(회차별), 잔금
3. **날짜 형식**: 반드시 `YYYY. M. D` 형식 (예: 2025. 6. 30)
4. **기간 처리**:
   - 단일 날짜: 시작일과 종료일 동일
   - 기간: 시작일과 종료일 다름
5. **지능형 날짜 계산** 필요:
   - "계약시" → 계약체결 일정과 동일
   - "계약 후 30일 이내" → 계약 종료일 + 30일
   - "입주지정일" → 입주예정월 마지막 날

## 📝 예시 패턴 인식

### 패턴 1: 표 형식
```
구분              일시
특별공급          2024년 10월 14일
1순위 청약        2024년 10월 15일
당첨자 발표       2024년 10월 22일
```

### 패턴 2: 납부 일정
```
구분        납부일           비율
계약금(1차) 계약시           10%
계약금(2차) 계약 후 30일     10%
중도금1회   2025.2.10       10%
잔금        입주지정일       60%
```

### 패턴 3: 상대적 날짜 계산

**계약체결 기간**: 2024. 11. 4 ~ 2024. 11. 6

- "계약시" → 2024. 11. 4 ~ 2024. 11. 6 (계약체결과 동일)
- "계약 후 30일 이내" → 2024. 12. 6 (계약 종료일 + 30일)

**입주예정월**: 2025년 6월

- "입주지정일" → 2025. 6. 30 (6월 마지막 날)
