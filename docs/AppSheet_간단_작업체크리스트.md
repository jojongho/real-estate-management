# AppSheet 고객 정보 추가 - 간단 체크리스트 ⚡

**상황**: 매물 등록 폼에 고객 정보만 추가하면 됨
**작업 시간**: 10-15분

---

## 🎯 3단계로 끝내기

### ✅ 1단계: Ref 필드 추가 (3분)
```
□ AppSheet 열기
□ Data → Tables → 아파트매물 선택
□ Columns 탭 클릭

□ "고객" 컬럼이 있는가?
   ✅ 있음 → 2단계로
   ❌ 없음 → 아래 작업 수행:
   
   □ + Add column 클릭
   □ Column name: 고객
   □ Type: Ref 선택
   □ Source table: 고객DB 선택
   □ Label column: 고객성함 (자동 선택됨)
   □ Save 클릭
```

### ✅ 2단계: Is Part Of 활성화 (2분)
```
□ 방금 만든 "고객" 필드 클릭
□ 아래로 스크롤
□ "Relationships" 섹션 찾기
□ "Is Part Of?" 체크박스 클릭 ✅
□ Save
```

→ 이제 매물 입력 중에 고객 정보도 함께 입력할 수 있음!

### ✅ 3단계: 폼에 표시하기 (5분)
```
□ UX 메뉴 클릭
□ Views 클릭
□ 아파트매물 Form 찾아서 선택

□ 왼쪽 필드 목록에서 "고객" 찾기
□ "고객" 필드를 원하는 위치로 드래그

권장 위치:
┌─────────────────────────────┐
│ 매물 정보 섹션              │
│ - 단지명                    │
│ - 동                        │
│ - 호                        │
│ - 타입                      │
│ - 분양가 (자동)             │
│ - 발코니 (자동)             │
│ - 옵션 선택                 │
│ ────────────────────────────│ ← 여기!
│ 👤 고객 정보 섹션           │
│ - 고객 (← 이 필드 여기로!)  │
│ - 거래유형                  │
│ - 거래상태                  │
│ - 기타...                   │
└─────────────────────────────┘

□ 저장하기 (Save)
```

---

## 🧪 테스트하기 (3분)

```
□ Preview 버튼 클릭 (우측 상단)
□ "+ 새 매물 추가" 클릭
□ 단지명, 동, 호, 타입 입력
□ 분양가 확인 ✅
□ 옵션 선택 확인 ✅
□ "고객" 필드 클릭
   → 드롭다운에 기존 고객 목록 표시되는지 확인
   → "+ 새 고객 추가" 버튼 있는지 확인
□ "+ 새 고객 추가" 클릭
   → 고객 등록 폼이 열리는지 확인
□ 고객 정보 입력 (이름, 연락처 등)
□ 저장
□ 원래 매물 폼으로 돌아오는지 확인
□ 매물 저장
□ 고객DB에서 새 고객이 등록되었는지 확인 ✅
```

## 🎉 완료!

이제 매물 등록할 때마다:
- ✅ 고객을 선택하거나
- ✅ 새 고객을 바로 추가할 수 있음!

---

## 🚨 문제 해결

### "고객" 필드가 안 보여요
→ UX → Views → Form → 필드 목록에서 "고객" 찾아서 Show를 ON으로 변경

### "새 고객 추가" 버튼이 없어요
→ Data → Columns → 아파트매물 → 고객 → "Allow add from form" 체크

### 고객 정보가 저장이 안 돼요
→ 권한 확인: UX → Security → Roles에서 추가 권한 설정

---

## 🎯 추가 개선 (선택사항)

나중에 여유 있을 때:

**섹션 구분** (폼을 더 보기 좋게):
```
□ Form 편집 → + Add section 클릭
□ "매물 정보" 섹션
□ "고객 정보" 섹션
□ 각 필드를 섹션으로 드래그
```

**필수 필드 표시**:
```
□ 각 필드 클릭 → "Required" 체크
```

**고객 연락처 자동 표시**:
```
□ Data → Columns → 아파트매물 → + Add column
□ Column name: 고객연락처
□ Type: Virtual
□ App formula: [고객].[연락처]
```
